# 📐 Task 1 – Low-Fidelity Wireframes

## Overview
This task involved the creation of low-fidelity wireframes for the **Aura Interiors** mobile application. The wireframes establish the fundamental layout, navigation structure, and user flow, serving as the blueprint for the final interface design.

## Objective
The primary objective was to define the application's information architecture and ensure an intuitive user experience before applying visual design elements.

## Key Screens
- Splash / Logo Screen
- Login & Signup
- Style Selection (Minimal, Luxury, Traditional)
- Home Dashboard
- AI Design Assistant (Rey)
- Design Preview
- Budget Calculation 
- Furniture Selection
- Designer & Shop Booking
- Wishlist
- User Profile

## Tools Used
- **Figma** – For wireframing and layout planning.
