# 🎨 Task 2 – High-Fidelity UI Design

## 📌 Overview
This task focuses on the development of a high-fidelity user interface for the **Aura Interiors** mobile application. The design transforms the previously created wireframes into a visually appealing and interactive experience, ensuring usability, consistency, and adherence to modern UI/UX principles.

## 🎯 Objective
- To create a premium and user-friendly mobile interface.
- To apply consistent branding, typography, and color schemes.
- To integrate AI-driven features for interior design and budget estimation.
- To deliver an interactive prototype that demonstrates the complete user flow.

## ✨ Key Features
- **Rey – AI Design Assistant:** Generates personalized interior design suggestions based on user preferences.
- **Budget calculation:** Provides detailed cost estimations and budget breakdowns.
- **Style Selection:** Users can choose from Minimal, Luxury, and Traditional themes.
- **Manual Design Editor:** Enables users to create custom layouts using drawing tools and image uploads.
- **Furniture & Decor Selection:** Allows users to explore and select suitable interior elements.
- **Designer & Shop Booking:** Facilitates real-world implementation by connecting users with professionals.
- **Wishlist & Profile:** Enables users to save favorites and manage personal information.

## 🎨 UI/UX Principles Applied
- **Visual Hierarchy:** Clear emphasis on important elements.
- **Consistency:** Uniform color palette, typography, and component usage.
- **User-Centered Design:** Focused on intuitive navigation and usability.
- **Accessibility:** Readable text and well-structured layouts.
- **Minimalism:** Clean and uncluttered interface with a premium aesthetic.

## 🎨 Design Style
The interface adopts a warm brown and beige color palette complemented by soft shadows and subtle glassmorphism effects, creating an elegant and modern user experience.
