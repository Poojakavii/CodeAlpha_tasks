# 🏡 Aura Interiors – AI-Powered Interior Design App

## 📌 Overview
**Aura Interiors** is a UI/UX design project for a premium mobile application that helps users create personalized interior designs using artificial intelligence. The app enables users to explore different design styles, generate room layouts, estimate budgets, and connect with interior designers and furniture shops for real-world implementation.

## 🎯 Objectives
- To design an intuitive and user-friendly mobile interface.
- To integrate AI-driven design and budget estimation features.
- To provide a seamless experience from ideation to implementation.

## ✅ Tasks Completed

### 🔹 Task 1: Low-Fidelity Wireframes
- Developed wireframes to define the layout, navigation, and user flow.
- Focused on usability and information architecture.
- Served as the foundation for the final UI design.

### 🔹 Task 2: High-Fidelity UI Design
- Created a visually appealing mobile interface using Figma.
- Applied a warm brown and beige color palette with glassmorphism and soft shadows.
- Ensured adherence to core UI/UX principles such as consistency, visual hierarchy, and accessibility.

## 🤖 Key Features
- **Rey – AI Design Assistant:** Generates personalized interior design ideas.
- **Budget calculator:** Provides detailed cost estimation based on user requirements.
- **Style Selection:** Options for Minimal, Luxury, and Traditional themes.
- **Manual Design Editor:** Tools for users to create custom layouts.
- **Furniture & Decor Selection:** Explore and choose suitable items.
- **Designer & Shop Booking:** Connect with professionals for implementation.
- **Wishlist:** Save favorite designs and products.
- **User Profile:** Manage personal information and preferences.

## 🛠️ Tools Used
- **Figma** – UI/UX design and prototyping
- **GitHub** – Project documentation and version control
